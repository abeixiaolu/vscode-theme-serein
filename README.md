## Match with System

<!--eslint-skip-->

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "Serein Light",
  "workbench.preferredDarkColorTheme": "Serein Dark",
}
```
